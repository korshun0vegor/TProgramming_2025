# rpgsaga

## Перед запуском

1. Установите **[Node.js](https://nodejs.org/)** (рекомендуется LTS, например 20.x).
2. В корне проекта выполните установку зависимостей:

```bash
npm install
```

## Запуск и проверки

| Команда                | Назначение                                                   |
| ---------------------- | ------------------------------------------------------------ |
| `npm run start`        | Запуск примера (`tsx src/index.ts`)                          |
| `npm test`             | Запуск тестов из папки `tests/`                              |
| `npm run build`        | Проверка типов TypeScript без сборки файлов (`tsc --noEmit`) |
| `npm run lint`         | Проверка кода ESLint                                         |
| `npm run format`       | Автоформатирование Prettier                                  |
| `npm run format:check` | Проверка форматирования без изменений                        |

## Как работать в проекте

- **`src/`** — ваш код (точка входа для запуска — `src/index.ts`; экспортируйте функции и типы для использования в тестах).
- **`tests/`** — тесты Vitest; импортируйте модули из `src/` (как в `tests/sum.test.ts`).
- После изменений имеет смысл по очереди запускать **`npm run build`**, **`npm test`** и при необходимости **`npm run lint`**.

Минимальный пример в репозитории — функция `sum` и демонстрационный вывод в консоль при `npm run start`.
